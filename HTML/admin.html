Cấu trúc User Collection: 
{
  username: string,
  email: string,
  avatar: string,
  coverImage: string,
  bio: string,
  role: "user"|"moderator"|"admin",
  status: "active"|"inactive",
  followers: number,
  following: number,
  createdAt: timestamp,
  lastActive: timestamp
}
Lưu ý cho Khôi:
  • Sử dụng Firebase Auth UID làm document ID
  • Gọi createUser() sau khi đăng ký thành công
  • Role mặc định là "user"
  • Status mặc định là "active"
  • Chỉ dùng followers & following (không friends)
